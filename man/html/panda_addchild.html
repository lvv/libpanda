panda_addchild

Name

panda_addchild -- add an object to the object tree

Synopsis

 #include<panda/constants.h>
 #include<panda/functions.h>
 void panda_addchild (panda_object *parent, panda_object *child);

DESCRIPTION

PANDA INTERNAL. Once an object has been created with panda_newobject() it is added to the object tree with this call. This ensures it is written out to disc via panda_writeobject() when panda_close() is called.

RETURNS

None

EXAMPLE

#include<panda/constants.h> #include<panda/functions.h> panda_pdf *document; panda_object *obj, *obj2; panda_init(); document = panda_open("filename.pdf", "w"); obj = panda_newobject(document, panda_object_normal); obj2 = panda_newobject(document, panda_object_normal); panda_addhild(obj, obj2);

SEE ALSO

panda_newobject, panda_freeobject, panda_writeobject, panda_traverseobjects
