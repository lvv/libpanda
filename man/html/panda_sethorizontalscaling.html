panda_sethorizontalscaling

Name

panda_sethorizontalscaling -- set the horizontal scaling of text

Synopsis

 #include<panda/constants.h>
 #include<panda/functions.h>
 void panda_sethorizontalscaling (panda_pdf *output, double amount);

DESCRIPTION

Set the horizontal scaling factor of the text in percent.

RETURNS

Nothing

EXAMPLE

#include<panda/constants.h> #include<panda/functions.h> panda_pdf *output; panda_init(); output = panda_open("output.pdf", "w"); panda_sethorizontalscaling(output, 42.3);
