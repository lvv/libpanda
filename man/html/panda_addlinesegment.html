panda_addlinesegment

Name

panda_addlinesegment -- add a straight segment to the line shape we are drawing

Synopsis

 #include<panda/constants.h>
 #include<panda/functions.h>
 void panda_addlinesegment (panda_page * target, int x, int y);

DESCRIPTION

Add a point to the shape we are currently drawing with a straight line between the current cursor location and (x,y).

RETURNS

Nothing

EXAMPLE

#include<panda/constants.h> #include<panda/functions.h> panda_pdf *document; panda_page *page; panda_init(); document = panda_open("filename.pdf", "w"); page = panda_newpage (document, panda_pagesize_a4); panda_setlinestart (page, 100, 200); panda_addlinesegment (page, 200, 200);
