panda_xmalloc

Name

panda_xmalloc -- allocate some memory

Synopsis

 #include<panda/constants.h>
 #include<panda/functions.h>
 void *panda_xmalloc (size_t size);

DESCRIPTION

PANDA INTERNAL. Allocate some memory, performing all of the error checking so we don't have to elsewhere.

RETURNS

A void * to the memory allocated.

EXAMPLE

#include<panda/constants.h> #include<panda/functions.h> char *memory; memory = (char *) panda_xmalloc(42);
