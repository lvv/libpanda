panda_putc

Name

panda_putc -- output a single character to the PDF file

Synopsis

 #include<panda/constants.h>
 #include<panda/functions.h>
 void panda_printf (panda_pdf * file, char c);

DESCRIPTION

PANDA INTERNAL. This function call behaves just like putc(), but the output goes to a PDF file, instead of to stdout.

RETURNS

Nothing

EXAMPLE

his is an internal function which will only be needed by those playing deeply with Panda itself, so I won't provide an example.

SEE ALSO

panda_print, panda_printf, panda_streamprintf
