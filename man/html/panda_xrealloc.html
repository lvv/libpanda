panda_xrealloc

Name

panda_xrealloc -- allocate some memory, resizing it if already exists

Synopsis

 #include<panda/constants.h>
 #include<panda/functions.h>
 void *panda_xmalloc (void *now, size_t size);

DESCRIPTION

PANDA INTERNAL. Allocate some memory, performing all of the error checking so we don't have to elsewhere. Resize the memory if it already exists.

RETURNS

A void * to the memory allocated.

EXAMPLE

#include<panda/constants.h> #include<panda/functions.h> char *memory; memory = (char *) panda_xrealloc(memory, 42);
